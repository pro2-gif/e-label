# 구현 계획서: QR코드 이미지 SVG 다운로드 기능 추가

기존 PNG 포맷 다운로드 기능 외에 고화질 벡터 포맷인 SVG 형식으로도 QR코드를 다운로드할 수 있는 버튼을 추가하고 다운로드 로직을 구현합니다.

## 해결 대상 요구사항
- "기존의 qr코드 이미지 다운로드 png를 svg로도 다운받을 수 있게 수정해줘"
- QR코드가 삽입되는 인쇄물이나 패키지 디자인 작업 시 고해상도 배율 조절이 가능한 벡터 포맷(SVG)의 QR코드가 필요합니다.
- 사용 중인 구글 차트 QR API(`api.qrserver.com`)의 포맷 변환 옵션(`&format=svg`)을 이용해 실시간으로 SVG 파일을 생성하고 즉시 다운로드할 수 있도록 기능을 확장합니다.

## 변경 제안 내용

### [웹 메이커]

#### [MODIFY] [index.html](file:///c:/Users/user/Desktop/e-label/index.html)
- **UI 레이아웃 변경**:
  - 기존 세로형 `[QR 코드 이미지 다운로드 (.png)]` 단일 버튼 대신, 가로로 나란히 정렬되는 **[PNG 다운로드]** 및 **[SVG 다운로드]** 버튼 그룹을 추가합니다.
  - 가독성과 심미성을 고려하여 SVG 다운로드 버튼은 프리미엄한 초록색 그라데이션 테마로 디자인을 다르게 적용합니다.
- **다운로드 이벤트 추가 및 변경**:
  - `#btn-download` (PNG 다운로드) 버튼 클릭 시 기존 PNG URL 발급 및 다운로드 연동 유지.
  - `#btn-download-svg` (SVG 다운로드) 버튼 클릭 시 `&format=svg` 파라미터가 포함된 QR API 주소를 Fetch하여 `.svg` 확장자로 다이렉트 다운로드 연동 로직 구현.
  - `updateQrDisplay` 호출 시 `currentQrUrl` 외에 `currentQrSvgUrl` 전역 변수를 추가하여 다운로드 클릭 시 즉각 사용할 수 있도록 변경합니다.
