# 구현 계획서: 전자라벨 뷰어 데이터 매칭 및 영어 번역 오류 수정

구글 시트 연동 주소 체계를 개선하고, 네트워크 등의 문제로 인한 데이터 로딩 실패 시 표시되는 예비(Fallback) 데이터의 전성분, 소비자 상담 번호, 제조업자 영문 주소를 실제 구글 시트와 100% 동일하게 동기화합니다.

## 해결 대상 문제

1. **구글 시트 연동 실패 및 Fallback 작동**:
   - `pub?output=csv` 방식은 구글 시트에서 "웹에 게시" 설정을 별도로 켜야만 작동합니다. 미설정 시 `401 Unauthorized` 또는 `404` 에러가 발생하여 계속해서 예비(Fallback) 데이터가 표시되고 있었습니다.
   - 공유 링크(`export?format=csv`) 형식으로 주소를 변경하여, 시트가 전체 공개되어 있는 한 100% 정상적으로 데이터를 긁어오도록 조치합니다.

2. **클리닉스 화이트 및 하이드로 데이터 불일치 (전성분 축약 문제)**:
   - 예비 데이터의 전성분이 일부분만 노출되는 문제를 구글 시트 실제 원본에 있는 전체 성분 텍스트로 대체하여 완벽히 일치시킵니다.

3. **영어(EN) 모드에서 제조업자/주소가 한글로 노출되는 현상**:
   - 예비 데이터의 화이트와 하이드로 제품에는 제조업자 한글 주소만 들어가 있어서 영문 버전이 매칭되지 못했습니다. 한글 뒤에 개행(`\n`)을 포함해 영문 주소를 함께 삽입함으로써, 영어 모드 전환 시 영문 주소가 정상적으로 출력되도록 수정합니다.

4. **소비자 상담 번호 오기재**:
   - 전체적으로 소비자 상담 번호가 시트의 진짜 번호(`1661-2383, 02-868-1921`)와 일치하도록 예비 데이터를 전면 수정합니다.
